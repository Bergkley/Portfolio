Project 1

GitHub

Descrição

Aplicação Web Responsiva Full Stack de E-Commerce B2B (Business-to-Business). O projeto foi estruturado para fornecer visões e funcionalidades distintas para cada tipo de usuário. O administrador (Organização Atacadista) têm acesso a recursos avançados de gerenciamento (cadastro de promoções, produtos, transportadoras, etc) e é ele que vende os produtos para os usuários comuns (Organizações Compradoras), enquanto os usuários comuns desfrutam de uma experiência de navegação e visualização simplificados. O sistema contém 41 páginas diferentes (urls) e 71 rotas de acesso a API (endpoints), todas desenvolvidas manualmente!
